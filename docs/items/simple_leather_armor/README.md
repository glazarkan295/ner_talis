# Нер-Талис — готовый архив: Простая кожаная броня

Архив завершён для текущего набора.

## Состав

1. Простой кожаный шлем — `simple_leather_helmet`, слот `helmet`.
2. Простой кожаный пояс — `simple_leather_belt_reworked`, слот `belt`.
3. Простой кожаный нагрудник — `simple_leather_chestpiece`, слот `chest`.
4. Простые кожаные ботинки — `simple_leather_boots`, слот `boots`.
5. Простые кожаные штаны — `simple_leather_pants`, слот `legs`.
6. Простые кожаные перчатки — `simple_leather_gloves`, слот `gloves`.

Каждый предмет имеет 3 варианта качества: обычный, необычный и редкий.

## Общие правила

- Базовой цены покупки нет.
- Цена продажи: 150 / 300 / 500 медных монет.
- Крафт: Кожевенная мастерская `leatherwork`.
- 1 предмет = 1 минута крафта.
- Уровень назначается при создании/нахождении предмета.
- Сила эффектов зависит от уровня созданного предмета.
- Шанс качества при создании: обычный 50%, необычный 40%, редкий 10%.

## Важные уточнения

- Пояс получил id `simple_leather_belt_reworked`, чтобы не конфликтовать со старым legacy-поясом.
- Нагрудник нормализован как `chest`.
- Штаны нормализованы как `legs`.
- Редкие перчатки имеют в пуле эффектов `Точность`.
