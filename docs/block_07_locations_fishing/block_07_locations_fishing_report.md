## Как работает рыбалка на пристани

Игрок заходит: `Портовый квартал → Пристань → Рыбалка на пристани`.

На экране рыбалки есть кнопка `Забросить удочку`.

При забросе проверяется:

- находится ли игрок в зоне рыбалки;
- есть ли в инвентаре `Удочка рыбака`;
- хватает ли 2 энергии.

Если всё подходит:

- списывается 2 энергии;
- бросается категория улова;
- выбирается конкретный предмет из таблицы;
- предмет добавляется в инвентарь с источником `Рыбалка на пристани`;
- игрок получает текст с категорией улова, названием предмета, количеством и остатком энергии.

## Как работают водные события локаций

При обычном поиске в локациях может выпасть событие `waterside_loot`.

Кнопки события:

- `Осмотреть и забрать`;
- `Пропустить`.

При осмотре игрок получает один предмет из таблицы водной находки для текущей локации.
