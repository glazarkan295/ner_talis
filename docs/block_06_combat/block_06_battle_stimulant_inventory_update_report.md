# Нер-Талис — корректировка боевого стимулятора

## Изменение

Боевой стимулятор переведён из боевого подсумка в обычное использование из инвентаря.

Теперь:

- `battle_stimulant` не отображается в подсумке боя;
- предмет остаётся расходником и используется через инвентарь/профиль;
- при использовании создаётся активный эффект `effect_battle_stimulant`;
- длительность эффекта: 30 минут;
- эффект при входе в бой даёт:
  - +30% к урону навыков;
  - +20% к максимуму Духа;
  - +20% к максимуму Маны;
- повторное использование обновляет активный эффект;
- для будущей логики зависимости при использовании увеличивается счётчик `battle_stimulant_addiction.level`;
- старые боевые зелья и метательные предметы остаются в подсумке.

## Тексты/поведение

В подсумке боевой стимулятор не показывается. Если игрок хочет принять стимулятор, он делает это из инвентаря. После применения игрок получает сообщение:

```text
Боевой стимулятор принят. Эффект активен 30 минут.
```

В бою эффект подхватывается автоматически, если активный эффект ещё не истёк.
