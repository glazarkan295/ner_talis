# Блок 7 — Обыкновенный лес: таймеры, энергия и тексты поиска

Источник: `ner_talis-main-51-block7-ordinary-forest-loot-corrected.zip`.

## Что сделано

Проверены и зафиксированы правила поиска в локации **Обыкновенный лес**:

- базовый расход поиска: **2 энергии**;
- базовое время при полной энергии: **30 секунд**;
- при низкой положительной энергии время поиска растёт, но не превышает **5 минут**;
- при **0 энергии** поиск занимает **10 минут** и тратит **0 энергии**;
- таймер поиска хранит `location_id = ordinary_forest`;
- отмена через `Назад` сбрасывает поиск и не выдаёт событие;
- `Проверить таймер` завершает только истёкший таймер;
- текст начала поиска показывает локацию, время, расход энергии, остаток энергии и HP;
- текст завершения поиска показывает локацию перед событием/ловушкой/боем.

## Изменения в данных

В `data/ordinary_forest.json` добавлен блок `search_rules`:

```json
{
  "version": "ordinary_forest_search_timing_v1",
  "energy_cost": 2,
  "base_time_seconds": 30,
  "positive_energy_max_time_seconds": 300,
  "zero_energy_time_seconds": 600
}
```

Входной текст Обыкновенного леса дополнен правилами энергии и времени:

- поиск тратит 2 энергии;
- базовое время — 30 секунд;
- при низкой энергии максимум 5 минут;
- при 0 энергии — 10 минут без траты энергии;
- возможные события: сухое дерево, грибы, речушка, небольшая нора, лесная ловушка или бой.

## Изменения в runtime

В `external_location_service.py` обновлён fallback-текст `ORDINARY_FOREST_TEXT`, чтобы даже при недоступном JSON отображались актуальные правила поиска.

Дополнительно исправлены служебные ответы для неактуальных/отсутствующих таймеров: теперь кнопки и зона ответа берутся из текущей локации, а не принудительно из Холмистых лугов.

## Сопутствующая проверка дропа

Во время запуска тестов обнаружено, что runtime-каталог мобов Обыкновенного леса всё ещё содержал старые названия точечных трофеев (`Волчий клык`, `Кабаний клык`, `Медвежий клык`, `Медвежий коготь`, `Жир зверя`, `Рог оленя`). Это было приведено к уже утверждённой канонической схеме:

- клыки → `Простой клык`;
- коготь медведя → `Простой коготь`;
- жир зверя → `Кусок жира`;
- рог оленя → `Оленьи рога`.

Обновлён тест `test_ordinary_forest_mobs_drop.py`, чтобы он проверял точные `item_id` в активных таблицах дропа, а не подстроки внутри `deer_antlers`.
