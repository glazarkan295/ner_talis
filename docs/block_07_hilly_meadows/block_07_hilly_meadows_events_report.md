# Блок 7 — Холмистые луга: пересборка событий локации

Источник: `ner_talis-main-51-block7-hilly-meadows-resources-updated.zip`.

## Что изменено

- Пересобрана таблица событий `data/hilly_meadows.json` до версии `hilly_meadows_events_v2`.
- Сумма весов событий приведена к 100.
- `waterside_loot` не возвращён и отсутствует в таблице событий.
- Добавлен блок `event_flow` с описанием логики каждого события.
- Добавлен блок `event_texts` для текстов обнаружения, чтобы тексты Холмистых лугов можно было менять через JSON.
- `create_search_event(...)` теперь берёт тексты событий Холмистых лугов из `data/hilly_meadows.json` с безопасными fallback-текстами.
- Варианты блика теперь читаются из `glint_variants` локации.
