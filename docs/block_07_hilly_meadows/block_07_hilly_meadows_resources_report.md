# Блок 7 — Холмистые луга: список ресурсов и находок

## Что изменено

- Для Холмистых лугов добавлен явный список ресурсов/находок в `data/hilly_meadows.json`.
- Runtime поиска больше не держит награды луга только в жёстко зашитых списках: награды берутся из `resource_find_tables` с безопасными fallback-значениями.
- `waterside_loot` не возвращён: водная/береговая находка остаётся убранной из Холмистых лугов и Обыкновенного леса.

## Итоговые группы Холмистых лугов

### Травы / алхимические ингредиенты

- Луговая мята — 1–2, вес 20.
- Серебристая ромашка — 1–2, вес 20.
- Жёлтый клевер — 1–2, вес 20.
- Горная полынь — 1–2, вес 20.
- Луговой корень — 1–2, вес 20.

### Ягоды

- Сладкая луговая ягода — 2–5, вес 50.
- Терпкая синяя ягода — 2–5, вес 50.

### Камни и руда

- Обычный камень — 1–3, вес 92.
- Кусок медной руды — 1–2, вес 5.
- Кусок железной руды — 1–2, вес 3.

### Блик выше по склону

- Железный лом — 1, вес 70.
- Старый нож — 1, вес 30.

### Следы ниже по склону

- Медные монеты — 1–10, вес 75.
- Серебряная монета — 1, вес 2.
- Ничего — вес 23.

## Замечание

Состав сделан стартовым и безопасным: редкие морские/водные находки, рыба, моллюски и сундучки не добавлены в Холмистые луга, потому что рыбалка теперь вынесена на пристань.
