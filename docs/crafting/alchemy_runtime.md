# Алхимия runtime

Алхимия подключена по системе без уровней изучения ингредиентов:

- игрок видит только название и базовое описание ингредиента;
- скрытые роли, теги, сила, токсичность, нестабильность и стабильность используются только во внутренних формулах;
- отдельный журнал провалов не используется;
- выбор основы, ингредиентов, катализаторов, стабилизаторов и действий идёт через нумерованные списки и ввод чисел в чат;
- действия выбираются одной строкой через пробел, например `1 8 5`;
- при превышении лимита действий показывается текст: `Вы выбрали слишком много действий для выбранных ингредиентов.`

Внутренние формулы учитывают:

- уровень алхимии;
- стадию;
- скрытую силу ингредиентов;
- токсичность;
- нестабильность;
- стабильность;
- количество действий;
- пороги допуска и гарантию успеха для младших стадий.
