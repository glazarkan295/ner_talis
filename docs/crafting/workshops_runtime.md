# Ремесленные мастерские runtime

Интегрированы рабочие мастерские Селдара:

- Плавильня — список рецептов сразу, кнопки `Крафт №N`.
- Кузница — разделы `Оружие`, `Броня`, `Заготовки`, `Рецепты`.
- Кожевенная мастерская — разделы `Броня`, `Заготовки`, `Рецепты`.
- Ювелирная мастерская — разделы `Кольца`, `Ожерелья`, `Рецепты`.
- Алхимическая мастерская — `Создать по рецепту`, `Эксперимент`, `Журнал рецептов`.

Общие правила:

- В списках рецептов показываются номера, название результата и отметка `✅`/`❌` по наличию ресурсов.
- В кнопках используются короткие подписи `Крафт №1`, `Крафт №2`, чтобы было удобно в Telegram/VK.
- При выборе рецепта показывается краткая карточка результата, ресурсы, время и сколько можно создать.
- При нажатии `Создать` игрок вводит количество числом в чат.
- Время ремесленного создания — 1 минута за 1 создаваемый предмет/партию: 1 предмет = 1 минута, 5 предметов = 5 минут.
- После завершения таймера результат добавляется через общий `inventory_service`, поэтому работают стаки, доп. слоты и ограничения инвентаря.
