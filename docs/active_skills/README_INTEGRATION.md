# Интеграционный архив активных навыков Нер-Талис

Архив подготовлен для подключения новой системы активных и пассивных навыков в проект.

## Основные файлы

- `ner_talis_active_skills_paths_0_10000.json` — полный каталог навыков, правил, порогов и текстов.
- `ner_talis_active_skills_paths_0_10000.md` — читаемое описание системы.
- `dispatcher_stone_texts.json` — тексты Распорядителя и камня для обработчиков ратуши.
- `dispatcher_stone_texts.md` — те же тексты в удобном виде.
- `integration_notes_0_10000.md` — технические заметки по подключению.

## Что нужно подключить в коде

1. Хранить у игрока блок развития навыков: `branch`, `main_path`, `secondary_path`, выбранные навыки, заблокированные варианты `choice_group`, вложенные очки модификаторов.
2. При достижении 10 уровня отправлять игроку сообщение из `level_10_notification`.
3. При достижении 100 уровня отправлять сообщение из `level_100_notification`, если основной путь уже выбран, а дополнительный ещё нет.
4. В ратуше добавить кнопку/действие распорядительного камня.
5. При нажатии камня выполнять проверки в порядке из `stone_check_priority`.
6. При выборе навыка сохранять один выбранный `id`, а два других из того же `choice_group` блокировать.
7. Пассивные навыки выводить в профиль в отдельный слот `Пассивные навыки`; в боевые кнопки их не добавлять.
8. Для дополнительного пути проверять лимит: `secondary_path_level <= floor(main_path_level * 0.6)`.

## Мягкая формула модификаторов

```text
Бонус % = floor(30 × ln(1 + Очки / 30))
```

В интерфейсе процент показывать с двумя знаками после запятой: `+0,00%`.
