# Отчёт синхронизации колчанов

Проверено активных навыков: **94**.

Навыков, требующих боеприпасы и колчан: **4**.

## Правило

Лук требует стрелы в экипированном колчане для стрел, арбалет требует болты в экипированном колчане для болтов. Проверка колчана и боеприпасов выполняется перед тратой духа/маны и перед броском попадания. Боеприпас списывается из колчана при попытке применения навыка, даже если атака промахнулась.

## Навыки

| Навык | Требования |
|---|---|
| Прицельный выстрел (`spirit_aimed_shot`) | `bow` → Стрела для лука ×1 в Пустой колчан для стрел лука; `crossbow` → Болт для арбалета ×1 в Пустой колчан для болтов арбалета |
| Ливень стрел (`spirit_arrow_rain`) | `bow` → Стрела для лука ×3 в Пустой колчан для стрел лука |
| Пробивающий болт (`spirit_piercing_bolt`) | `crossbow` → Болт для арбалета ×1 в Пустой колчан для болтов арбалета |
| Метка охотника (`spirit_hunter_mark`) | `bow` → Стрела для лука ×1 в Пустой колчан для стрел лука; `crossbow` → Болт для арбалета ×1 в Пустой колчан для болтов арбалета |
