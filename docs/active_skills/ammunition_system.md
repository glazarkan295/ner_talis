# Система боеприпасов для активных навыков

## Главное правило

Лук требует стрелы в экипированном колчане для стрел, арбалет требует болты в экипированном колчане для болтов. Проверка колчана и боеприпасов выполняется перед тратой духа/маны и перед броском попадания. Боеприпас списывается из колчана при попытке применения навыка, даже если атака промахнулась.

## Связка оружия, боеприпаса и колчана

| Оружие | Боеприпас | Где должен лежать боеприпас | Колчан |
|---|---|---|---|
| Лук (`bow`) | Стрела для лука (`arrow_for_bow`) | В экипированном колчане для стрел | Пустой колчан для стрел лука (`arrow_quiver_empty`) |
| Арбалет (`crossbow`) | Болт для арбалета (`bolt_for_crossbow`) | В экипированном колчане для болтов | Пустой колчан для болтов арбалета (`bolt_quiver_empty`) |

## Порядок проверки при применении навыка

1. Проверить, что навык разрешён для текущего оружия.
2. Если текущее оружие — лук, проверить экипированный колчан для стрел.
3. Если текущее оружие — арбалет, проверить экипированный колчан для болтов.
4. Проверить, что в нужном колчане есть достаточно стрел или болтов.
5. Только после этого проверять дух/ману и выполнять бросок попадания.
6. Боеприпас списывается из колчана при попытке применения навыка, даже если атака промахнулась.

## Навыки, которые требуют боеприпасы и колчан

| Навык | Оружие | Расход |
|---|---|---|
| Прицельный выстрел (`spirit_aimed_shot`) | `bow`, `crossbow` | `bow`: 1 × Стрела для лука; `crossbow`: 1 × Болт для арбалета |
| Ливень стрел (`spirit_arrow_rain`) | `bow` | `bow`: 3 × Стрела для лука |
| Пробивающий болт (`spirit_piercing_bolt`) | `crossbow` | `crossbow`: 1 × Болт для арбалета |
| Метка охотника (`spirit_hunter_mark`) | `bow`, `crossbow`, `staff`, `dagger` | `bow`: 1 × Стрела для лука; `crossbow`: 1 × Болт для арбалета |

## Важное примечание

Свободные стрелы и болты могут лежать в инвентаре как предметы, но стрелковые навыки тратят только боеприпасы, загруженные в подходящий экипированный колчан.
