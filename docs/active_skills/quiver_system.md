# Система колчанов

## Главное правило

Для хранения и боевого использования стрел и болтов нужны специальные колчаны.

- Лук использует только стрелы, загруженные в **колчан для стрел**.
- Арбалет использует только болты, загруженные в **колчан для болтов**.
- Пустой колчан можно экипировать, но стрелковый навык не выполнится, пока в нём нет нужного количества боеприпасов.
- Свободные стрелы/болты в инвентаре не считаются доступными для навыка, пока не загружены в подходящий колчан.

## Предметы

| Предмет | ID | Тип | Вместимость | Для чего нужен |
|---|---|---|---:|---|
| Пустой колчан для стрел лука | `arrow_quiver_empty` | Контейнер боеприпасов | 30 | Хранит стрелы для лука |
| Пустой колчан для болтов арбалета | `bolt_quiver_empty` | Контейнер боеприпасов | 20 | Хранит болты для арбалета |
| Стрела для лука | `arrow_for_bow` | Боеприпас | — | Тратится навыками с луком |
| Болт для арбалета | `bolt_for_crossbow` | Боеприпас | — | Тратится навыками с арбалетом |

## Рекомендуемая структура состояния игрока

```json
{
  "equipped_quivers": {
    "arrow_quiver": {
      "item_id": "arrow_quiver_empty",
      "slot": "arrow_quiver",
      "capacity": 30,
      "ammo_item_id": "arrow_for_bow",
      "ammo_count": 12
    },
    "bolt_quiver": {
      "item_id": "bolt_quiver_empty",
      "slot": "bolt_quiver",
      "capacity": 20,
      "ammo_item_id": "bolt_for_crossbow",
      "ammo_count": 5
    }
  }
}
```

## Сообщения бота

- Нет колчана для лука: «Для навыков с луком нужен колчан для стрел.»
- Нет колчана для арбалета: «Для навыков с арбалетом нужен колчан для болтов.»
- В колчане нет стрел: «В колчане нет нужного количества стрел.»
- В колчане нет болтов: «В колчане нет нужного количества болтов.»
