# Каталог активных навыков Нер-Талис

Готовый каталог для интеграции. Активные навыки **не открываются напрямую за уровень игрока**. Уровень 10 даёт только системный доступ к выбору ветви в Селдаре, в Ратуше, у Распорядительного камня.

## Оружие, боеприпасы и колчаны

`weapon_requirements` всегда хранится как список. Проверка идёт по правилу `any_of`: навык можно использовать, если экипирован хотя бы один подходящий тип оружия из списка.

Луки и арбалеты используют боеприпасы через специальные колчаны: лук тратит стрелы из экипированного колчана для стрел, арбалет тратит болты из экипированного колчана для болтов. Для навыков с несколькими видами оружия колчан и боеприпас требуются только если навык применяется с луком или арбалетом.

- `Прицельный выстрел`: `bow`, `crossbow` — с луком тратит стрелу из колчана для стрел, с арбалетом тратит болт из колчана для болтов.
- `Ливень стрел`: `bow` — тратит 3 стрелы из колчана для стрел.
- `Пробивающий болт`: `crossbow` — тратит 1 болт из колчана для болтов.

## Стартовые нейтральные действия

### Обычный удар (`neutral_basic_strike`)

- **Ветка:** neutral
- **Ресурс:** none
- **Категория:** attack / basic_melee
- **Тип урона:** physical
- **Цель:** single_enemy
- **Оружие:** sword, dagger, staff, axe, hammer
- **Профиль характеристик:** Сила 55%, Ловкость 25%, Восприятие 20%
- **Коэффициент роли:** 0.45
- **Базовая стоимость ресурса:** 0
- **Откат:** 0 ход(а/ов)
- **Открытие:**
  - тип: `starter_action`
  - не открывается напрямую за уровень игрока
  - требования:
    - создан профиль игрока
    - выдаётся при создании персонажа, не относится к Ветви Духа или Ветви Маны

**Эффект:**
  - Простая физическая атака. Выдаётся при создании персонажа, не открывает ветки и не прокачивается как основная ветвь.

**Текст в бою:** Вы наносите обычный удар по выбранной цели.
