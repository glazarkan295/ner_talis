# Блок 7 — Обыкновенный лес: мобы и дроп

Источник: `ner_talis-main-51-block7-ordinary-forest-events-rebuilt.zip`.

## Что сделано

Настроен боевой каталог локации **Обыкновенный лес** и вынесены актуальные таблицы мобов/дропа в `data/ordinary_forest.json`.

Добавлены/обновлены блоки:

- `battle_mobs_version = ordinary_forest_mobs_drop_v1`
- `battle_rank_chances`
- `battle_mob_tables`
- `battle_drop_tables`
- `battle_drop_rules`

Runtime `pve_battle_service.py` обновлён: выбор мобов Обыкновенного леса теперь идёт через отдельную таблицу весов `ORDINARY_FOREST_MOB_WEIGHTS`, а не через равновероятный `rng.choice`.

## Мобы Обыкновенного леса

В локации настроены 4 моба:

| mob_id | Название | Роль |
|---|---|---|
| `forest_wolf` | Волк | атакующий зверь |
| `angry_deer` | Разъярённый олень | атакующий зверь |
| `forest_boar` | Кабан | защищающийся зверь |
| `forest_bear` | Медведь | тяжёлый защищающийся зверь |

## Правила дропа по рангу

- обычный моб — шанс и количество по таблице;
- усиленный моб — шанс добычи ×1.1, количество не повышается;
- элитный моб — шанс добычи ×1.5, количество повышается: минимум ×1.5, максимум ×1.7 с округлением вверх.

## Item ID

Добыча Обыкновенного леса выдаётся через item_id:

- общие трофеи: `simple_hide`, `raw_meat`, `simple_tendon`;
- лесные трофеи: `wolf_fang`, `deer_antler`, `boar_tusk`, `animal_fat`, `bear_claw`, `bear_fang`.

## Добавленные тесты

Добавлен тест `test_ordinary_forest_mobs_drop.py`, который проверяет:

- наличие таблиц мобов и дропа в `ordinary_forest.json`;
- наличие всех 4 мобов в runtime-каталоге;
- наличие item_id-маппинга для всех предметов дропа;
- отсутствие медведя в обычном бою;
- элитный бой только с медведем;
- уровни мобов Обыкновенного леса в диапазоне 10–60;
- наличие элитного ранга в ранговых шансах локации.
