# Блок 7 — Холмистые луга: таймеры, энергия и тексты поиска

Источник: `ner_talis-main-51-block7-hilly-meadows-loot-items-corrected.zip`.

## Что проверено и исправлено

Проверены таймеры поиска, расход энергии и тексты поиска в локации `Холмистые луга`.

### Таймеры поиска

Актуальные правила:

- базовое время поиска при полной энергии: **30 секунд**;
- при низкой положительной энергии время поиска увеличивается, но не выше **5 минут**;
- при `0` энергии поиск занимает **10 минут**;
- активный поиск создаёт `active_timer` с типом `search`, `location_id=hilly_meadows`, `seconds` и `ends_at`;
- кнопка `Проверить таймер` завершает только истёкший таймер;
- кнопка `Назад` во время поиска отменяет таймер и не выдаёт событие.

### Энергия

Актуальные правила:

- один поиск в Холмистых лугах тратит **2 энергии**;
- если энергии меньше 2, тратится фактически доступное количество;
- при `0` энергии поиск разрешён, но тратит `0` энергии и длится 10 минут;
- поля `energy` и `current_energy` синхронизируются;
- предупреждения об энергии `≤50` и `≤10` сохранены.

### Тексты поиска

Обновлён входной текст Холмистых лугов: теперь прямо указаны расход энергии, базовое время, лимит 5 минут при низкой энергии и 10 минут при нулевой энергии.

Стартовый текст поиска теперь показывает:

- начало поиска;
- локацию;
- время поиска;
- потраченную энергию;
- остаток энергии;
- HP игрока.

Текст завершения поиска теперь показывает локацию перед найденным событием/боем/ловушкой.

### Дополнительные исправления

- В `data/hilly_meadows.json` добавлен блок `search_rules` с явными правилами времени и энергии.
- Исправлены старые многострочные строки в `resolve_glint_event(...)`, чтобы `external_location_service.py` стабильно компилировался.
- В стартовый текст PVE-боя добавлена строка локации, чтобы бой из Холмистых лугов явно отображал `Холмистые луга`.
- Обновлён старый тест количества предметов в `items_hilly_meadows.json`: после канонизации трофеев там 25 локальных предметов, а общие трофеи мобов лежат в `items_starting_mob_loot.json` без дублей.
