# Блок 7 — Обыкновенный лес: пересборка событий

Источник: `ner_talis-main-51-block7-ordinary-forest-burrow-corrected.zip`.

## Что изменено

- Таблица событий Обыкновенного леса пересобрана до версии `ordinary_forest_events_v2`.
- Сумма весов событий приведена к 100.
- `waterside_loot` не возвращён и явно указан как исключённое событие.
- Добавлен блок `event_flow` с описанием логики событий.
- Добавлен блок `event_texts`, чтобы тексты обнаружения можно было менять через JSON.
- `create_search_event(...)` теперь берёт тексты событий Обыкновенного леса из JSON с fallback-текстами.
