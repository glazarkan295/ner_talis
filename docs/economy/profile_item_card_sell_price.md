# Изменение профиля: цена продажи в карточке предмета

Внесено изменение для отображения цены продажи прямо в карточке предмета в профиле игрока.

## Что изменено

1. `ner_talis_game_project/site_api.py`
   - при нормализации предмета для профиля добавляются поля:
     - `canSell`
     - `sellPriceCopper`
     - `sellPriceText`
   - если предмет продаётся и имеет `sell_price_copper`, в карточку передаётся текст вида `25 медных`;
   - если предмет явно не продаётся, передаётся `не продаётся`.

2. `web/src/components/player-profile/PlayerProfile.jsx`
   - в модальной карточке предмета добавлена строка:
     - `Цена продажи — <значение>`
   - поддерживаются оба формата поля цены:
     - `sell_price_copper`
     - `sellPriceCopper`

3. `web/src/components/player-profile/profileMockData.js`
   - добавлены примеры цены продажи для тестового отображения профиля.

## Где отображается

Цена показывается при открытии карточки предмета в профиле, рядом с основными данными предмета:

- Тип
- Слот
- Количество
- Цена продажи
