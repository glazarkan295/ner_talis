# Полное ТЗ для Claude Code: Telegram-клавиатура, сворачивание меню и команды `/hide_menu` / `/menu`

## 1. Задача

В Telegram-боте Нер-Талис нужно исправить UX нижней игровой клавиатуры на мобильных устройствах.

Сейчас на телефонах игроку неудобно выходить из чата через системную кнопку телефона **«Назад»**. Нижняя клавиатура бота постоянно активна, из-за чего Telegram-клиент может сначала сворачивать/разворачивать клавиатуру, фокусировать поле ввода или удерживать нижнюю панель, а не сразу выходить из чата.

Нужно сделать правильную работу нижней клавиатуры Telegram:

```text id="8bsmlw"
1. Обычное сворачивание/разворачивание — через стандартную кнопку Telegram.
2. Полное скрытие нижней клавиатуры — через команду /hide_menu.
3. Возврат нижней клавиатуры — через команду /menu.
```

---

## 2. Важное техническое уточнение

Кнопка с иконкой клавиатуры/квадратиков рядом с полем ввода — это **стандартная кнопка Telegram-клиента**.

Бот не может:

```text id="g138mj"
нарисовать эту кнопку вручную
изменить её иконку
переместить её
перехватить системную кнопку телефона Назад
управлять анимацией нижней панели Telegram
```

Но бот может сделать так, чтобы эта стандартная кнопка Telegram появлялась и работала корректно.

Для этого нужно правильно использовать:

```python id="fj3xcy"
ReplyKeyboardMarkup
ReplyKeyboardRemove
```

---

## 3. Требуемое поведение

### 3.1. Обычное игровое меню

Когда игрок вызывает:

```text id="lkd9un"
/start
/menu
```

или открывает главное меню, бот должен отправить нижнюю клавиатуру через:

```python id="03tbm0"
ReplyKeyboardMarkup
```

После этого Telegram сам показывает стандартную кнопку сворачивания/разворачивания нижней клавиатуры рядом с полем ввода.

Ожидаемое поведение на телефоне:

```text id="q7i8a5"
Игрок отправляет /menu
→ бот показывает нижнее игровое меню
→ Telegram показывает стандартную кнопку клавиатуры
→ игрок нажимает стандартную кнопку Telegram
→ нижняя клавиатура сворачивается
→ игрок нажимает стандартную кнопку Telegram снова
→ нижняя клавиатура разворачивается
```

---

### 3.2. Полное скрытие меню

Команда:

```text id="kov88i"
/hide_menu
```

должна полностью убрать нижнюю reply-клавиатуру через:

```python id="jqoiw6"
ReplyKeyboardRemove()
```

После полного скрытия стандартная кнопка Telegram для разворачивания клавиатуры может пропасть. Это нормально, потому что активная reply-клавиатура удалена.

Чтобы вернуть клавиатуру, игрок должен отправить:

```text id="kyt6y7"
/menu
```

---

### 3.3. Возврат меню

Команда:

```text id="ijherm"
/menu
```

должна вернуть основную нижнюю игровую клавиатуру через:

```python id="n83hxp"
ReplyKeyboardMarkup
```

---

## 4. Основная настройка ReplyKeyboardMarkup

Главное Telegram-меню должно создаваться примерно так:

```python id="w3zvr2"
ReplyKeyboardMarkup(
    keyboard=main_keyboard,
    resize_keyboard=True,
    one_time_keyboard=False,
    is_persistent=False
)
```

Обязательные требования:

```text id="95tadz"
resize_keyboard=True
one_time_keyboard=False
is_persistent=False
```

Пояснение:

```text id="etq1t8"
resize_keyboard=True — Telegram подгоняет размер клавиатуры.
one_time_keyboard=False — меню не исчезает после каждого нажатия.
is_persistent=False — меню не становится навязчиво постоянным на телефоне.
```

Нельзя использовать по умолчанию:

```python id="4xbipr"
one_time_keyboard=True
is_persistent=True
```

---

## 5. Команды

Добавить или исправить команды Telegram-бота.

### 5.1. `/menu`

Назначение:

```text id="hu2aap"
Открыть / вернуть нижнее игровое меню.
```

Поведение:

1. сбросить флаг скрытого меню;
2. отправить сообщение:

```text id="90oxrx"
⌨️ Игровое меню открыто.
```

3. приложить основную нижнюю клавиатуру через `ReplyKeyboardMarkup`.

---

### 5.2. `/hide_menu`

Назначение:

```text id="x6iyp6"
Полностью скрыть нижнее игровое меню.
```

Поведение:

1. установить флаг скрытого меню;
2. отправить сообщение:

```text id="b5hgw1"
⌨️ Игровое меню скрыто.

Чтобы вернуть его, отправьте /menu.
```

3. приложить:

```python id="y0eq5v"
ReplyKeyboardRemove()
```

---

### 5.3. `/start`

Команда `/start` должна:

1. начать или продолжить игру;
2. сбросить флаг скрытого меню;
3. вернуть основную нижнюю клавиатуру.

Если игрок ранее скрывал меню через `/hide_menu`, команда `/start` должна снова показать меню.

---

### 5.4. Дополнительные текстовые варианты

Желательно поддержать обычный текст без `/`:

```text id="xmd1p4"
меню
открыть меню
вернуть меню
```

Они должны работать как `/menu`.

Текстовые варианты скрытия можно не добавлять, чтобы случайно не скрывать меню. Основной способ скрытия:

```text id="x8uu86"
/hide_menu
```

## 7. Состояние скрытого меню

Добавить состояние для Telegram-пользователя:

```python id="uj5o52"
telegram_menu_hidden = True / False
```

Можно хранить:

```text id="a0j9ey"
в player state
в runtime session
в отдельном Telegram UI state
```

Главное требование: бот должен помнить, что пользователь сам скрыл меню.

Когда пользователь вызывает:

```text id="msu3wj"
/hide_menu
```

установить:

```python id="0z2dyi"
telegram_menu_hidden = True
```

Когда пользователь вызывает:

```text id="4jtaq3"
/menu
/start
```

установить:

```python id="a6xrv0"
telegram_menu_hidden = False
```

## 8. Не возвращать клавиатуру самовольно

Если:

```python id="7i2gxx"
telegram_menu_hidden == True
```

бот не должен автоматически прикладывать `ReplyKeyboardMarkup` к обычным игровым или информационным сообщениям.

Клавиатуру можно вернуть только при явном действии игрока:

```text id="lwhjd1"
/menu
/start
меню
открыть меню
вернуть меню
```

Исключение допустимо только для критического игрового сценария, где без меню нельзя продолжить. Но даже в таком случае лучше отправить текст:

```text id="4e6db7"
Чтобы продолжить через нижнее меню, отправьте /menu.
```

## 9. Главное меню без кнопки «Скрыть меню»

Не добавлять в нижнюю клавиатуру отдельную кнопку:

```text id="w8j32b"
🔽 Скрыть меню
```

Вместо неё используется команда:

```text id="l5ljym"
/hide_menu
```

Причины:

```text id="mkzm2u"
нижняя клавиатура не засоряется технической кнопкой
стандартная кнопка Telegram уже сворачивает/разворачивает меню
полное скрытие нужно редко и лучше через команду
```

---

## 10. Разница между стандартной кнопкой Telegram и `/hide_menu`

### 10.1. Стандартная кнопка Telegram

Назначение:

```text id="v2v0m2"
Обычное сворачивание/разворачивание нижней клавиатуры.
```

Поведение:

```text id="3ov3kp"
Клавиатура остаётся активной.
Игрок может свернуть её.
Игрок может развернуть её обратно той же стандартной кнопкой Telegram.
```

Для этого бот должен сохранять активную `ReplyKeyboardMarkup`.

---

### 10.2. Команда `/hide_menu`

Назначение:

```text id="v6prfl"
Полное удаление нижней клавиатуры.
```

Поведение:

```text id="2xhzuy"
Клавиатура удаляется через ReplyKeyboardRemove.
Стандартная кнопка Telegram может пропасть.
Вернуть клавиатуру можно через /menu.
```

## 12. Правила использования reply-клавиатуры

### 12.1. Где использовать ReplyKeyboardMarkup

Использовать для:

```text id="cu2cs4"
главного меню
часто используемой игровой навигации
базовых игровых действий
```

Пример:

```text id="9zxe8s"
☰ Меню
👤 Профиль
🎒 Инвентарь
🗺 Локация
🏙 Город
⚔️ Бой
```

---

### 12.2. Где использовать InlineKeyboardMarkup

Для временных действий лучше использовать inline-кнопки под сообщением:

```text id="8tcyga"
подтверждение действия
выбор предмета
выбор количества
выбор длительности
покупка/продажа
подтверждение удаления
открытие профиля/сайта
одноразовые действия
```

Это уменьшит зависимость от нижней клавиатуры и улучшит UX на телефонах.

---

### 12.3. Где не отправлять ReplyKeyboardMarkup

Не прикладывать нижнюю клавиатуру к каждому сообщению, особенно если:

```text id="rj4yky"
игрок скрыл меню через /hide_menu
сообщение информационное
сообщение является результатом действия
сообщение приходит как уведомление
сообщение не требует немедленного выбора из нижнего меню
```

## 14. Поведение при WebApp / профиле

Если Telegram-бот открывает профиль игрока, сайт или админ-панель через WebApp/ссылку, проверить:

```text id="qxu8o6"
после закрытия WebApp бот не должен самовольно возвращать persistent-клавиатуру
/profile не должен ломать состояние telegram_menu_hidden
/menu должен возвращать клавиатуру после скрытия
```

Кнопки профиля можно делать через inline-кнопку:

```text id="2eq7gt"
Открыть профиль
```

а не через навязчивую нижнюю reply-кнопку, если так удобнее.

## 15. Поведение в личных сообщениях и группах

Правка касается Telegram-бота в личном чате.

Если бот находится в группах/общих чатах, не показывать игровую нижнюю клавиатуру там, где игра должна работать только в личном чате.

В группах бот не должен засорять чат reply-клавиатурами.

## 19. Ручная проверка на Android

Проверить на телефоне Android:

1. открыть личный чат с Telegram-ботом;
2. отправить `/start`;
3. убедиться, что нижняя клавиатура появилась;
4. убедиться, что рядом с полем ввода есть стандартная кнопка Telegram для сворачивания/разворачивания клавиатуры;
5. нажать стандартную кнопку Telegram;
6. нижняя клавиатура должна свернуться;
7. нажать стандартную кнопку Telegram снова;
8. нижняя клавиатура должна развернуться;
9. отправить `/hide_menu`;
10. нижняя клавиатура должна полностью исчезнуть;
11. нажать системную кнопку телефона «Назад»;
12. Telegram должен вести себя как обычный чат;
13. снова открыть чат;
14. отправить `/menu`;
15. нижняя клавиатура должна появиться снова;
16. выполнить несколько игровых действий;
17. если меню было скрыто, оно не должно возвращаться само без `/menu`.

---

## 20. Ручная проверка на iPhone

Проверить:

1. `/menu` показывает нижнюю клавиатуру;
2. стандартная кнопка Telegram сворачивает/разворачивает клавиатуру, если клиент Telegram это поддерживает;
3. `/hide_menu` полностью скрывает клавиатуру;
4. `/menu` возвращает клавиатуру;
5. игровые действия не ломаются.

---

## 21. Ручная проверка на Desktop Telegram

Проверить:

1. `/menu` показывает reply-клавиатуру;
2. `/hide_menu` убирает reply-клавиатуру;
3. `/menu` возвращает reply-клавиатуру;
4. команда не ломает ввод текста.

---

## 22. Критерии готовности

Исправление считается готовым, если:

1. В Telegram на телефоне появляется стандартная кнопка сворачивания/разворачивания нижней клавиатуры.
2. Игрок может свернуть нижнюю клавиатуру стандартной кнопкой Telegram.
3. Игрок может развернуть нижнюю клавиатуру стандартной кнопкой Telegram.
4. Команда `/hide_menu` полностью скрывает нижнюю клавиатуру через `ReplyKeyboardRemove`.
5. Команда `/menu` возвращает нижнюю клавиатуру через `ReplyKeyboardMarkup`.
6. Команда `/start` также возвращает нижнюю клавиатуру.
7. После `/hide_menu` бот не возвращает клавиатуру самовольно.
8. Главное меню не использует `one_time_keyboard=True`.
9. Главное меню не использует `is_persistent=True` по умолчанию.
10. На Android после `/hide_menu` системная кнопка «Назад» работает как в обычном Telegram-чате.
11. Игровые кнопки Telegram продолжают работать.
12. Reply-клавиатура не отправляется в групповые чаты, если игра там не должна работать.
13. VK-бот и VK-клавиатура не затронуты.
14. Тесты покрывают `/menu`, `/hide_menu`, `/start`, состояние `telegram_menu_hidden` и параметры клавиатуры.

## 23. Краткая формулировка задачи

Настроить нижнюю Telegram-клавиатуру Нер-Талис так, чтобы мобильный Telegram сам показывал стандартную кнопку сворачивания/разворачивания клавиатуры. Для полного скрытия меню добавить команду `/hide_menu` через `ReplyKeyboardRemove`, а для возврата меню — команду `/menu` через `ReplyKeyboardMarkup`. После `/hide_menu` бот не должен самовольно возвращать клавиатуру, пока игрок явно не вызовет `/menu` или `/start`.
