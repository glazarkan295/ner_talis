# 06. Prompt for Claude Code

Используй этот пакет как единое ТЗ по проекту Нер-Талис.

## Что сделать

1. Прочитай `00_CLAUDE_CODE_START_HERE.md`.
2. Сначала исправь баги из `01_CODEX_CRITICAL_FIXES.md`, начиная с P1.
3. Затем реализуй фундамент админ-панели из `02_MASTER_IMPLEMENTATION_TZ.md`.
4. Для эффектов обязательно используй `04_EFFECTS_CONSTRUCTOR_TZ.md`.
5. Для всех загрузок изображений и медиа обязательно выполни `07_MEDIA_UPLOAD_REQUIREMENTS.md`.
6. После каждого блока сверяйся с `05_ACCEPTANCE_CHECKLIST.md`.

## Жёсткие правила

- Не ломай текущий runtime.
- Published-контент нельзя скрывать из игры простым draft-edit.
- Не допускай дублей предметов/монет/наград при retry.
- Не допускай потери outbox-сообщений.
- Не выдавай edit-токены read-only ролям.
- Не затирай ручные правки импортом без подтверждения.
- Все интерфейсные названия должны быть на русском.
- Игроку не показывай формулы.
- Все загружаемые изображения/видео/файлы сохраняй как файлы проекта.
- Загрузчик изображений должен принимать только PNG/JPG/WebP, проверять содержимое через Pillow, нормализовать файл, запрещать внешние URL для сообщений и создавать оптимизированные варианты по `07_MEDIA_UPLOAD_REQUIREMENTS.md`.

## Формат работы

- Делай изменения небольшими логическими коммитами.
- После каждого этапа запускай тесты/проверки.
- Если тестов нет, добавляй минимальные regression-тесты для критичных багов.
- В отчёте указывай:
  - что исправлено;
  - какие файлы изменены;
  - какие риски остались;
  - что проверено вручную.
