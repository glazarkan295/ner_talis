# Нер-Талис — пакет ТЗ для Claude Code

Дата подготовки: 2026-06-25

## Назначение пакета

Этот пакет собран для передачи в Claude Code.  
Он объединяет загруженные ТЗ, большой файл по эффектам и правки Codex.

## Как работать с пакетом

1. Сначала прочитать `00_CLAUDE_CODE_START_HERE.md`.
2. Затем выполнить срочные правки из `01_CODEX_CRITICAL_FIXES.md`.
3. После этого переходить к системной базе админ-панели из `02_MASTER_IMPLEMENTATION_TZ.md`.
4. Для эффектов использовать `04_EFFECTS_CONSTRUCTOR_TZ.md` как отдельный обязательный источник.
5. Для медиа-загрузок обязательно использовать `07_MEDIA_UPLOAD_REQUIREMENTS.md`.
6. Для приёмки использовать `05_ACCEPTANCE_CHECKLIST.md`.

Оригиналы сохранены в папке `original_sources/`.

## Важное правило

Если в исходных ТЗ есть повторяющиеся требования, считать их единым требованием.  
Если есть конфликт между новым конструктором и текущим runtime, приоритет:

1. безопасность и отсутствие дублирования наград/предметов;
2. сохранность данных;
3. live/draft/publish без поломки текущей игры;
4. корректность экономики;
5. удобство админ-панели;
6. новые игровые функции.
