# Нер-Талис — ТЗ: универсальные черты мобов, благословения и фазы боссов

Файл предназначен для Claude Code и будущей админ-панели конструкторов мобов, навыков, предметов, эффектов, черт, благословений и фаз боя.

Главное назначение файла: описать универсальные черты мобов по рангам, благословения, мировые черты, универсальные фазы боссов и требования к конструкторам. Контр-зелья и специальные ослабляющие составы в этот файл не включаются: они будут настраиваться отдельно в админ-панели как отдельная система.

---

# 1. Общая логика черт мобов

## 1.1. Ранги черт

В конструкторе мобов должны быть отдельные группы черт:

1. **Особые черты** — для мобов ранга `Особый`, а также для мини-боссов, боссов, рейдовых и мировых боссов по лимитам ранга.
2. **Элитные черты** — для мобов ранга `Элитный`, а также для мини-боссов, боссов, рейдовых и мировых боссов по лимитам ранга.
3. **Уникальные черты** — для мобов ранга `Уникальный`.
4. **Мировые черты** — для мобов ранга `Мировой босс`.

Черта — это не навык и не отдельная атака. Черта меняет поведение моба, его пассивные свойства, реакции, защиту, угрозу, награды, фазовую механику или взаимодействие с игроком.

## 1.2. Универсальность черт

Каждая черта должна подходить минимум 4–6 видам или категориям мобов. Черты не должны быть привязаны только к одному конкретному существу.

Примеры категорий мобов:

- звери;
- духи;
- нежить;
- мутанты;
- аномалии;
- големы и конструкты;
- насекомые;
- растения и грибы;
- водные и болотные существа;
- гуманоиды;
- бандиты и разбойники;
- древние стражи;
- элементали;
- проклятые существа;
- теневые сущности;
- демонические существа;
- боссы и мини-боссы.

## 1.3. Общие поля черты

```yaml
trait_id: string
trait_name: string
trait_rank: special / elite / unique / world
player_text: string
admin_description: string
applicable_mob_categories: list[string]
allowed_mob_ranks: list[string]
trigger: passive / battle_start / on_attack / on_receive_damage / on_turn_start / on_turn_end / on_death / phase_change
stack_rule: refresh / strongest_only / stack_limited / unique_only
max_stacks: integer
can_be_removed: boolean
formula: string
balance_notes: string
```

## 1.4. Общие правила баланса

- Особые черты должны слегка менять бой, но не превращать моба в мини-босса.
- Элитные черты должны заметно менять тактику боя.
- Уникальные черты должны делать конкретного моба узнаваемым и опасным, но не должны требовать отдельной системы контр-зелий внутри этого файла.
- Мировые черты должны менять весь бой, рейд, локацию или поведение группы игроков.
- Черты не должны создавать бесконечные цепочки эффектов.
- Ответный урон, периодический урон, отражение и шипы по умолчанию не запускают новые эффекты.
- Контр-зелья, ослабляющие составы, специальные масла, колбы и другие способы временного ослабления черт настраиваются отдельным модулем админ-панели, а не внутри этого файла.

# 2. Ранги мобов, лимиты навыков, черт и фаз

## 2.1. Общая таблица рангов

|---|---:|---:|---|---:|---:|
| Обычный | 2 | 0 | нет | 1 | 1 |
| Усиленный | 2 | 1 | нет | 1 | 1 |
| Особый | 3 | 1 | +1 особая черта | 1 | 1 |
| Элитный | 4–5 | 2–3 | +1 элитная черта | 1 | 1 |
| Мини-босс | 5–6 | 4 | +1–2 особые черты, +1 элитная черта | может иметь 2 фазы | 1 |
| Босс | 8–10 | 6 | +3 особые черты, +2 элитные черты | 2–3 фазы | 1–2 |
| Рейдовый босс | 15–20 | 10 | +5 особых черт, +3 элитные черты | 4–5 фаз | 1–3 |

## 2.2. Валидация рангов в конструкторе мобов

В конструкторе мобов при выборе ранга должны автоматически подставляться рекомендуемые лимиты, но админ может видеть предупреждение, если выходит за них.

```yaml

active_skill_limit_min: integer
active_skill_limit_max: integer
passive_skill_limit_min: integer
passive_skill_limit_max: integer
special_trait_limit_min: integer
special_trait_limit_max: integer
elite_trait_limit_min: integer
elite_trait_limit_max: integer
unique_trait_limit_min: integer
unique_trait_limit_max: integer
world_trait_limit: integer
phase_limit_min: integer
phase_limit_max: integer
active_skills_per_turn_min: integer
active_skills_per_turn_max: integer
validation_mode: strict / warning_only / manual_override
```

## 2.3. Важные правила по навыкам мобов

- Активные навыки моба выбираются из конструктора навыков с пометкой `для мобов` или `универсальный`.
- Пассивные навыки моба работают автоматически и не занимают действие в бою.
- Черта не считается пассивным навыком и хранится отдельно.
- Фаза не считается навыком, но может включать или выключать навыки, менять шансы, параметры и поведение моба.
- Для мирового босса лимит 1–6 активных навыков за ход должен зависеть от текущей фазы, состояния, условий боя и настроек навыков.

# 3. Особые черты для мобов ранга «Особый»

## 1. Крепкая шкура

**Для кого подходит:** звери, мутанты, насекомые, растения, болотные существа, гуманоиды в броне.

**Описание:** моб получает меньше физического урона.

**Формула:**

```text
final_physical_damage_taken = incoming_physical_damage * (1 - physical_reduction_percent / 100)
```

**Баланс:** 3–8% снижения физического урона.

---

## 2. Плотная оболочка

**Для кого подходит:** големы, конструкты, насекомые, мутанты, аномалии, древние стражи.

**Описание:** моб имеет дополнительную защиту от первого сильного удара.

**Формула:**

```text
if incoming_damage >= mob_max_hp * heavy_hit_threshold_percent / 100:
    final_damage = incoming_damage * (1 - shell_reduction_percent / 100)
```

**Баланс:** снижение 10–20% только против сильного удара.

---

## 3. Быстрая реакция

**Для кого подходит:** звери, бандиты, гуманоиды, насекомые, теневые существа, духи.

**Описание:** моб немного чаще уклоняется или снижает шанс попадания по себе.

**Формула:**

```text
final_mob_dodge = clamp(base_dodge + dodge_bonus_percent, 0, 35)
```

**Баланс:** +2–5% уклонения.

---

## 4. Острые когти

**Для кого подходит:** звери, мутанты, насекомые, демонические существа, нежить, болотные твари.

**Описание:** атаки моба имеют небольшой шанс вызвать кровотечение.

**Формула:**

```text
if hit_success and roll <= bleed_chance:
    apply_bleed(target, bleed_percent, duration_turns)
```

**Баланс:** шанс 5–12%, длительность 2–3 хода.

---

## 5. Едкая слюна

**Для кого подходит:** мутанты, болотные существа, насекомые, звери, аномальные твари, растения.

**Описание:** атака моба может немного снизить защиту цели.

**Формула:**

```text
if hit_success and roll <= acid_mark_chance:
    final_physical_defense = base_physical_defense * (1 - defense_penalty_percent / 100)
```

**Баланс:** шанс 5–10%, штраф защиты 3–7%.

---

## 6. Слабое ядовитое тело

**Для кого подходит:** насекомые, пауки, змеи, растения, болотные существа, мутанты.

**Описание:** при атаке в ближнем бою атакующий может получить слабое отравление.

**Формула:**

```text
if received_melee_attack and roll <= poison_contact_chance:
    apply_poison(attacker, poison_percent, duration_turns)
```

**Баланс:** шанс 5–10%, яд слабый.

---

## 7. Тяжёлое тело

**Для кого подходит:** големы, звери, мутанты, боссы-звери, конструкты, большие насекомые.

**Описание:** моб получает бонус к устойчивости к оглушению и отталкиванию.

**Формула:**

```text
final_control_chance = base_control_chance * (1 - control_resist_percent / 100)
```

**Баланс:** 10–25% сопротивления контролю.

---

## 8. Ночная охота

**Для кого подходит:** звери, нежить, бандиты, теневые существа, духи, мутанты.

**Описание:** в зоне тьмы, ночью или в подземелье моб получает бонус к точности и урону.

**Формула:**

```text
if location_has_tag("dark") or time_period == night:
    final_damage = base_damage * (1 + night_damage_bonus_percent / 100)
    final_accuracy = base_accuracy + night_accuracy_bonus_percent
```

**Баланс:** +3–8% урона, +3–6% точности.

---

## 9. Территориальная ярость

**Для кого подходит:** звери, духи, големы, аномалии, растения, нежить.

**Описание:** моб сильнее в своей основной локации.

**Формула:**

```text
if current_location in mob.home_locations:
    final_damage = base_damage * (1 + territory_damage_bonus_percent / 100)
```

**Баланс:** +5–10% урона или защиты только в родной локации.

---

## 10. Пугающий вид

**Для кого подходит:** нежить, мутанты, демонические существа, звери, аномалии, боссы.

**Описание:** при начале боя снижает точность или шанс критического удара игрока.

**Формула:**

```text
if battle_start and roll <= fear_chance:
    target_accuracy = target_accuracy - fear_accuracy_penalty
    target_crit_chance = max(0, target_crit_chance - fear_crit_penalty)
```

**Баланс:** эффект короткий, 1–2 хода.

---

## 11. Рваный удар

**Для кого подходит:** звери, мутанты, насекомые, бандиты, нежить, демонические существа.

**Описание:** атака с шансом накладывает лёгкую травму или кровотечение.

**Формула:**

```text
if hit_success and roll <= torn_hit_chance:
    apply_bleed_or_minor_trauma(target)
```

**Баланс:** шанс 5–12%, сила слабая.

---

## 12. Сопротивление стихии

**Для кого подходит:** элементали, духи, мутанты, аномалии, големы, растения.

**Описание:** моб получает меньше урона от одной выбранной стихии или типа зоны.

**Формула:**

```text
if incoming_damage_element == resistant_element:
    final_damage = incoming_damage * (1 - element_resist_percent / 100)
```

**Баланс:** 10–25% сопротивления.

---

## 13. Нестабильная плоть

**Для кого подходит:** мутанты, аномалии, нежить, демонические существа, проклятые существа, растения.

**Описание:** при получении урона моб с шансом получает случайный слабый баф или дебаф.

**Формула:**

```text
if received_damage and roll <= unstable_body_chance:
    apply_random_effect_from_pool(unstable_pool)
```

**Баланс:** пул эффектов должен быть ограничен слабыми эффектами.

---

## 14. Инстинкт выживания

**Для кого подходит:** звери, гуманоиды, бандиты, мутанты, насекомые, болотные существа.

**Описание:** при низком HP моб получает небольшой бонус к уклонению или побегу.

**Формула:**

```text
if mob_hp_percent <= survival_threshold_percent:
    mob_dodge = clamp(base_dodge + survival_dodge_bonus, 0, 35)
```

**Баланс:** порог 25–35% HP, бонус +3–8%.

---

## 15. Неприятная добыча

**Для кого подходит:** ядовитые существа, мутанты, нежить, болотные твари, растения, аномалии.

**Описание:** после победы над мобом есть риск получить слабый негативный эффект при сборе добычи.

**Формула:**

```text
if loot_collected and roll <= loot_risk_chance:
    apply_loot_penalty_effect(player)
```

**Баланс:** шанс 3–10%, эффект слабый или короткий.

---

# 4. Элитные черты для мобов ранга «Элитный»

## 16. Усиленная регенерация

**Для кого подходит:** звери, мутанты, нежить, растения, демонические существа, боссы.

**Описание:** моб восстанавливает здоровье каждый ход.

**Формула:**

```text
hp_restore = mob_max_hp * regen_percent / 100 + regen_flat
mob_current_hp = min(mob_max_hp, mob_current_hp + hp_restore)
```

**Баланс:** 0.5–2% max HP за ход.

---

## 17. Боевая адаптация

**Для кого подходит:** гуманоиды, древние стражи, аномалии, мутанты, боссы, уникальные мобы.

**Описание:** после нескольких ударов одного типа моб получает временное сопротивление этому типу урона.

**Формула:**

```text
if same_damage_type_hits >= adaptation_threshold:
    add_resistance(damage_type, adaptation_resist_percent)
```

**Баланс:** 5–20% сопротивления, ограничить максимум.

---

## 18. Ответный выпад

**Для кого подходит:** гуманоиды, звери, древние стражи, големы, демонические существа, элитные мобы.

**Описание:** после получения прямого удара моб с шансом отвечает атакой.

**Формула:**

```text
if received_direct_damage and roll <= counterattack_chance:
    counter_damage = base_attack_damage * counter_damage_percent / 100
```

**Ограничение:** не срабатывает от яда, поджога, кровотечения, шипов и отражения.

---

## 19. Давление элиты

**Для кого подходит:** элитные звери, бандиты, големы, нежить, демонические существа, древние стражи.

**Описание:** пока моб жив, игрок получает небольшой штраф к точности или уклонению.

**Формула:**

```text
player_accuracy = max(min_accuracy, base_accuracy - elite_pressure_accuracy_penalty)
player_dodge = max(0, base_dodge - elite_pressure_dodge_penalty)
```

**Баланс:** штраф 3–8%.

---

## 20. Бронебойные удары

**Для кого подходит:** големы, звери, гуманоиды, мутанты, древние стражи, боссы.

**Описание:** атаки моба частично игнорируют физическую защиту игрока.

**Формула:**

```text
bypass_damage = raw_damage * armor_penetration_percent / 100
normal_damage = raw_damage - bypass_damage
final_damage = bypass_damage + damage_after_defense(normal_damage, target_physical_defense)
```

**Баланс:** 5–15% пробоя.

---

## 21. Срывающий удар

**Для кого подходит:** гуманоиды, звери, големы, мутанты, боссы, демонические существа.

**Описание:** атака может временно заблокировать дополнительное действие игрока.

**Формула:**

```text
if hit_success and roll <= bonus_action_block_chance:
    block_bonus_action(target, duration_turns)
```

**Баланс:** шанс 5–15%, длительность 1 ход.

---

## 22. Угроза охотника

**Для кого подходит:** звери, бандиты, гуманоиды, демонические существа, мутанты, рейдовые враги.

**Описание:** моб выбирает ослабленную цель и наносит ей повышенный урон.

**Формула:**

```text
if target_hp_percent <= weak_target_threshold:
    final_damage = base_damage * (1 + weak_target_bonus_percent / 100)
```

**Баланс:** +10–25% урона по целям с низким HP.

---

## 23. Подавление лечения

**Для кого подходит:** нежить, демонические существа, аномалии, мутанты, боссы, проклятые существа.

**Описание:** пока моб жив, лечение игрока или группы работает слабее.

**Формула:**

```text
final_healing = base_healing * (1 - healing_suppression_percent / 100)
```

**Баланс:** 10–30% снижения лечения.

---

## 24. Нестабильное поле

**Для кого подходит:** аномалии, духи, элементали, древние стражи, демонические существа, боссы.

**Описание:** каждый несколько ходов на поле боя появляется случайный слабый эффект.

**Формула:**

```text
if current_turn % field_interval_turns == 0:
    apply_random_field_effect(field_effect_pool)
```

**Ограничение:** пул эффектов должен быть безопасным для баланса.

---

## 25. Усиление от ранения

**Для кого подходит:** звери, мутанты, боссы, демонические существа, нежить, големы.

**Описание:** чем меньше здоровья у моба, тем сильнее его атаки.

**Формула:**

```text
missing_hp_percent = 100 - mob_hp_percent
final_damage = base_damage * (1 + missing_hp_percent * damage_per_missing_hp_percent / 100)
```

**Баланс:** общий бонус ограничить 10–35%.

---

## 26. Защитная фаза

**Для кого подходит:** боссы, мини-боссы, големы, древние стражи, аномалии, уникальные мобы.

**Описание:** при низком HP моб временно усиливает защиту.

**Формула:**

```text
if hp_percent <= defense_phase_threshold:
    final_damage_taken = incoming_damage * (1 - defense_phase_reduction_percent / 100)
```

**Баланс:** снижение 10–30%, ограниченная длительность.

---

## 27. Призыв слабых помощников

**Для кого подходит:** боссы, некроманты, вожаки зверей, духи, растения, аномалии.

**Описание:** моб призывает слабых помощников через определённые интервалы.

**Формула:**

```text
if current_turn % summon_interval_turns == 0 and summon_count < max_summons:
    spawn_minion_from_pool(minion_pool)
```

**Баланс:** максимум 1–3 помощника за бой для элитных мобов.

---

## 28. Аура угрозы

**Для кого подходит:** боссы, элитные враги, демонические существа, нежить, древние стражи, аномалии.

**Описание:** снижает боевую эффективность игроков вокруг моба.

**Формула:**

```text
if mob_alive:
    player_damage = base_damage * (1 - threat_aura_damage_penalty_percent / 100)
```

**Баланс:** 3–10% снижения урона или точности.

---

## 29. Сильная природная защита

**Для кого подходит:** звери, големы, растения, насекомые, мутанты, болотные существа.

**Описание:** моб получает заметную защиту от выбранного набора эффектов: яд, кровотечение, огонь, контроль или травмы.

**Формула:**

```text
final_effect_chance = base_effect_chance * (1 - natural_resist_percent / 100)
```

**Баланс:** 20–50% сопротивления выбранным эффектам.

---

## 30. Разрушитель защиты

**Для кого подходит:** големы, звери, гуманоиды, демонические существа, боссы, мутанты.

**Описание:** атаки моба могут временно снижать защиту игрока.

**Формула:**

```text
if hit_success and roll <= defense_break_chance:
    target_physical_defense = target_physical_defense * (1 - defense_break_percent / 100)
```

**Баланс:** шанс 10–20%, штраф 5–15% на 2–3 хода.

# 5. Уникальные черты для мобов ранга «Уникальный»

## 31. Уникальная защита

**Для кого подходит:** уникальные звери, аномалии, древние стражи, големы, духи, демонические существа.

**Описание:** моб имеет особую защиту, которая делает его узнаваемым и меняет тактику боя.

**Формула:**

```text
final_damage_taken = incoming_damage * (1 - unique_defense_percent / 100)
```

**Баланс:** защита 20–50%, с настройкой эффективности против боссов и массовых атак.

---

## 32. Уникальная атака

**Для кого подходит:** уникальные звери, духи, аномалии, боссы, мутанты, проклятые существа.

**Описание:** моб имеет особую усиленную атаку, которая отличается от обычных активных навыков.

**Формула:**

```text
unique_attack_damage = base_attack_damage * (1 + unique_attack_bonus_percent / 100)
```

**Баланс:** бонус 15–50%, с откатом, шансом применения или фазовым условием.

---

## 33. Меняющаяся защита

**Для кого подходит:** аномалии, мутанты, древние стражи, духи, элементали, уникальные боссы.

**Описание:** защита моба меняет тип каждые несколько ходов. Игроку нужно следить за текстом боя и менять тактику.

**Формула:**

```text
if current_turn % switch_interval_turns == 0:
    current_defense_tag = random_from(defense_tag_pool)
```

**Пример:** огненная защита → каменная защита → зеркальная защита.

---

## 34. Зеркальный панцирь

**Для кого подходит:** аномалии, древние стражи, духи, магические конструкты, элементали, уникальные мобы.

**Описание:** часть магического урона отражается обратно в игрока.

**Формула:**

```text
reflected_damage = received_magic_damage * reflect_percent / 100
can_be_reflected = false
can_trigger_effects = false
```

**Баланс:** 5–20% отражения, не должно запускать цепочки эффектов.

---

## 35. Непробиваемая стойка

**Для кого подходит:** гуманоиды-мастера, древние стражи, големы, звери-вожаки, демонические существа, уникальные бойцы.

**Описание:** моб резко снижает входящий физический урон, пока стойка активна.

**Формула:**

```text
if stance_active:
    final_physical_damage_taken = incoming_physical_damage * (1 - stance_defense_percent / 100)
```

**Баланс:** стойка должна иметь длительность, откат или фазовое условие.

---

## 36. Проклятое присутствие

**Для кого подходит:** нежить, духи, демонические существа, проклятые звери, аномалии, древние стражи.

**Описание:** пока моб жив, игрок имеет шанс получить слабое проклятье или усиление уже активного проклятья.

**Формула:**

```text
if turn_start and roll <= curse_presence_chance:
    apply_random_curse_or_extend_existing(target)
```

**Баланс:** шанс 3–10% за ход или за несколько ходов.

---

## 37. Разлом пространства

**Для кого подходит:** аномалии, духи, древние стражи, магические сущности, уникальные боссы, мутанты.

**Описание:** моб искажает бой: может менять цель атаки, снижать точность, увеличивать стоимость действий или переносить игрока в неблагоприятную позицию.

**Формула:**

```text
if turn_start and roll <= spatial_distortion_chance:
    apply_spatial_effect(random_from(spatial_effect_pool))
```

**Баланс:** пул пространственных эффектов должен быть ограничен.

---

## 38. Пожирание ресурса

**Для кого подходит:** аномалии, нежить, демонические существа, духи, мутанты, древние стражи.

**Описание:** моб забирает у игрока ману, Дух или энергию при атаке.

**Формула:**

```text
resource_drained = min(target_current_resource, damage_dealt * drain_percent / 100 + flat_drain)
```

**Важно:** Дух — ресурс физических боевых навыков, стоек и приёмов.

---

## 39. Фазовая неуязвимость

**Для кого подходит:** аномалии, духи, древние стражи, демонические существа, уникальные боссы, элементали.

**Описание:** моб периодически становится почти неуязвимым или получает сильное снижение урона до конца фазы.

**Формула:**

```text
if phase_state == intangible:
    final_damage_taken = incoming_damage * intangible_damage_multiplier
```

**Баланс:** не делать полной неуязвимостью без понятного текста и условия окончания.

---

## 40. Легендарная ярость

**Для кого подходит:** уникальные звери, боссы, демонические существа, големы, мутанты, гуманоиды-вожди.

**Описание:** при низком HP моб переходит в опасное состояние: наносит больше урона, чаще применяет навыки, но может получать повышенный входящий урон.

**Формула:**

```text
if hp_percent <= rage_threshold:
    final_damage = base_damage * (1 + rage_damage_bonus_percent / 100)
    final_damage_taken = incoming_damage * (1 + rage_vulnerability_percent / 100)
```

**Баланс:** ярость должна быть опасной, но не бесконечной.

# 6. Мировые черты для мировых боссов

Мировая черта — редкое глобальное свойство мирового босса. Она должна менять не только параметры, но и правила боя, локации, рейда или мира на время события.

## 41. Давление мирового существа

**Для кого подходит:** мировые звери, древние стражи, аномалии, демонические существа, элементали, гигантские мутанты.

**Описание:** пока мировой босс жив, все участники боя получают небольшой штраф к точности, уклонению или восстановлению.

**Формула:**

```text
if world_boss_alive:
    player_accuracy = max(min_accuracy, base_accuracy - world_pressure_accuracy_penalty)
    player_dodge = max(0, base_dodge - world_pressure_dodge_penalty)
```

---

## 42. Сотрясение региона

**Для кого подходит:** големы, великаны, древние стражи, земные элементали, мутанты, рейдовые чудовища.

**Описание:** каждые несколько ходов босс вызывает региональный удар, который задевает всех участников.

**Формула:**

```text
if current_turn % quake_interval_turns == 0:
    deal_area_damage(all_players, boss_power * quake_power_percent / 100)
```

---

## 43. Мировая аура подавления

**Для кого подходит:** нежить, демонические существа, аномалии, духи, проклятые боссы, древние стражи.

**Описание:** ослабляет лечение, очищение или благословения игроков во всём бою.

**Формула:**

```text
final_healing = base_healing * (1 - world_healing_suppression_percent / 100)
final_cleanse_chance = base_cleanse_chance * (1 - world_cleanse_suppression_percent / 100)
```

---

## 44. Разрушение строя

**Для кого подходит:** мировые звери, полководцы, демонические существа, гуманоидные боссы, древние стражи, аномалии.

**Описание:** снижает эффективность групповых аур, рейдовых бонусов и защитных построений.

**Формула:**

```text
final_group_aura_power = base_group_aura_power * (1 - formation_break_percent / 100)
```

---

## 45. Поглощение мира

**Для кого подходит:** аномалии, демонические существа, нежить, древние сущности, мировые мутанты, тёмные элементали.

**Описание:** мировой босс периодически поглощает часть маны, Духа или энергии игроков.

**Формула:**

```text
resource_loss = target_max_resource * world_drain_percent / 100
current_resource = max(0, current_resource - resource_loss)
```

---

## 46. Зов прислужников

**Для кого подходит:** нежить, звери-вожаки, насекомые-ульи, демонические боссы, растения, древние стражи.

**Описание:** мировой босс призывает волны помощников в зависимости от фазы или таймера.

**Формула:**

```text
if current_turn % world_summon_interval == 0 and summoned_count < max_world_summons:
    spawn_minion_wave(world_minion_pool)
```

---

## 47. Нестабильная реальность

**Для кого подходит:** аномалии, духи, древняя магия, пространственные боссы, демонические существа, элементали.

**Описание:** периодически меняет правила боя: стоимость навыков, точность, силу лечения, шанс событий или порядок хода.

**Формула:**

```text
if current_turn % reality_shift_interval == 0:
    apply_random_world_battle_modifier(reality_modifier_pool)
```

---

## 48. Неистощимое тело

**Для кого подходит:** мировые звери, големы, растения, мутанты, демонические существа, древние стражи.

**Описание:** мировой босс восстанавливает часть здоровья, если игроки долго не наносят достаточно урона.

**Формула:**

```text
if damage_received_last_interval < required_damage_threshold:
    boss_hp = min(boss_max_hp, boss_hp + boss_max_hp * regen_percent / 100)
```

---

## 49. Катастрофический предел

**Для кого подходит:** мировые боссы всех категорий.

**Описание:** если бой затягивается слишком долго, босс начинает усиливаться по таймеру.

**Формула:**

```text
if battle_duration_turns >= soft_enrage_turn:
    boss_damage = base_damage * (1 + enrage_stack_count * enrage_bonus_per_stack / 100)
```

---

## 50. Мировая добыча

**Для кого подходит:** мировые боссы всех категорий.

**Описание:** награды после победы зависят от вклада игроков, стадии события, сложности и участия в фазах.

**Формула:**

```text
reward_score = damage_score + support_score + survival_score + objective_score
final_reward_tier = get_reward_tier(reward_score)
```

**Важно:** черта не должна только усиливать босса; она задаёт правила честного распределения награды.

# 7. Универсальные фазы для боссов

Эти фазы можно использовать для мирового босса, рейдового босса, босса и мини-босса. Фаза — это набор условий, модификаторов, доступных навыков, текстов и правил поведения. Фаза может включаться по HP, таймеру, числу ходов, смерти помощников, накоплению ресурса или выполнению игроками условия.

## 1. Вступительная фаза

**Для кого подходит:** мини-босс, босс, рейдовый босс, мировой босс.

**Описание:** начальная фаза боя. Босс использует базовый набор навыков и показывает игрокам главную механику.

**Формула:**

```text
if battle_start:
    boss_phase = intro
```

**Настройки:** доступно 30–50% активных навыков босса, низкая частота сильных атак.

---

## 2. Фаза проверки защиты

**Для кого подходит:** босс, рейдовый босс, мировой босс, мини-босс-защитник.

**Описание:** босс начинает наносить сильные удары, проверяя физическую и магическую защиту игроков.

**Формула:**

```text
if hp_percent <= phase_threshold:
    incoming_damage_to_players *= 1 + defense_check_bonus_percent / 100
```

---

## 3. Фаза давления

**Для кого подходит:** все боссы.

**Описание:** босс чаще атакует и сокращает безопасные окна для восстановления.

**Формула:**

```text
active_skills_per_turn = min(max_skills_per_turn, active_skills_per_turn + pressure_bonus_actions)
```

---

## 4. Фаза призыва

**Для кого подходит:** некроманты, звери-вожаки, растения, аномалии, демонические существа, древние стражи.

**Описание:** босс призывает помощников или волны врагов.

**Формула:**

```text
if phase_active and current_turn % summon_interval == 0:
    spawn_minion_wave(minion_pool)
```

---

## 5. Фаза защитного панциря

**Для кого подходит:** големы, звери с панцирем, древние стражи, насекомые, мутанты, мировые боссы.

**Описание:** босс получает сильную защиту, пока не закончится фаза или не выполнится условие.

**Формула:**

```text
final_damage_taken = incoming_damage * (1 - shell_phase_reduction_percent / 100)
```

---

## 6. Уязвимая фаза

**Для кого подходит:** все боссы.

**Описание:** после выполнения условия босс временно получает больше урона.

**Формула:**

```text
if boss_phase == vulnerable:
    final_damage_taken = incoming_damage * (1 + vulnerability_bonus_percent / 100)
```

---

## 7. Фаза ярости

**Для кого подходит:** звери, мутанты, демонические существа, гуманоиды-вожди, големы, боссы.

**Описание:** при низком HP босс наносит больше урона и чаще использует активные навыки.

**Формула:**

```text
if hp_percent <= rage_threshold:
    boss_damage = base_damage * (1 + rage_damage_bonus_percent / 100)
```

---

## 8. Фаза истощения игроков

**Для кого подходит:** нежить, аномалии, демонические существа, духи, проклятые боссы, мировые боссы.

**Описание:** босс начинает активно снижать ресурсы игроков: ману, Дух, энергию или восстановление.

**Формула:**

```text
resource_loss = target_max_resource * resource_drain_percent / 100
```

---

## 9. Фаза контроля поля

**Для кого подходит:** аномалии, магические боссы, древние стражи, духи, элементали, рейдовые боссы.

**Описание:** на поле боя появляются эффекты зоны: туман, огонь, мороз, тьма, нестабильность или ловушки.

**Формула:**

```text
if phase_active and current_turn % field_effect_interval == 0:
    apply_field_effect(random_from(field_effect_pool))
```

---

## 10. Фаза охоты на слабых

**Для кого подходит:** звери, убийцы, демонические существа, бандиты, мутанты, боссы-охотники.

**Описание:** босс чаще выбирает игроков с низким HP или негативными эффектами.

**Формула:**

```text
target_priority = choose_target(lowest_hp_or_most_debuffs)
final_damage = base_damage * (1 + weak_target_damage_bonus_percent / 100)
```

---

## 11. Фаза отражения

**Для кого подходит:** зеркальные стражи, аномалии, древние магические боссы, духи, элементали.

**Описание:** часть урона возвращается атакующим. Игроки должны осторожнее использовать мощные атаки.

**Формула:**

```text
reflected_damage = received_damage * reflect_percent / 100
can_be_reflected = false
```

---

## 12. Фаза запрета восстановления

**Для кого подходит:** нежить, проклятые боссы, демонические существа, аномалии, мировые боссы.

**Описание:** босс временно ослабляет лечение, регенерацию и очищение.

**Формула:**

```text
final_healing = base_healing * (1 - heal_block_percent / 100)
final_regeneration = base_regeneration * (1 - regen_block_percent / 100)
```

---

## 13. Фаза разлома

**Для кого подходит:** пространственные аномалии, духи, древние стражи, демонические существа, мировые боссы.

**Описание:** босс искажает порядок действий, цели, стоимость навыков или позиции.

**Формула:**

```text
if phase_active and roll <= rift_effect_chance:
    apply_random_rift_effect(rift_effect_pool)
```

---

## 14. Фаза накопления силы

**Для кого подходит:** все боссы.

**Описание:** босс накапливает заряд. Если игроки не прервут или не переживут механику, происходит сильная атака.

**Формула:**

```text
charge_stacks += 1
if charge_stacks >= required_charge_stacks:
    use_powerful_skill()
```

---

## 15. Фаза распада брони

**Для кого подходит:** големы, бронированные звери, древние стражи, насекомые, мини-боссы и боссы.

**Описание:** при снижении HP босс теряет защиту, но начинает наносить больше урона.

**Формула:**

```text
boss_defense = base_defense * (1 - armor_decay_percent / 100)
boss_damage = base_damage * (1 + armor_decay_damage_bonus_percent / 100)
```

---

## 16. Фаза восстановления

**Для кого подходит:** растения, мутанты, нежить, демонические существа, мировые боссы, рейдовые боссы.

**Описание:** босс пытается восстановить здоровье или снять с себя негативные эффекты.

**Формула:**

```text
if phase_active:
    boss_hp = min(boss_max_hp, boss_hp + boss_max_hp * phase_regen_percent / 100)
```

---

## 17. Фаза испытания урона

**Для кого подходит:** рейдовые боссы, мировые боссы, боссы подземелий.

**Описание:** игрокам нужно нанести достаточно урона за ограниченное число ходов, иначе босс усилится.

**Формула:**

```text
if damage_in_phase < required_damage:
    apply_boss_enrage_stack()
```

---

## 18. Фаза испытания выживания

**Для кого подходит:** рейдовые боссы, мировые боссы, проклятые боссы, аномалии.

**Описание:** босс временно усиливает давление, а задача игроков — пережить фазу.

**Формула:**

```text
survival_turns_left -= 1
if survival_turns_left <= 0:
    switch_next_phase()
```

---

## 19. Финальная фаза

**Для кого подходит:** босс, рейдовый босс, мировой босс.

**Описание:** последняя опасная фаза. Босс использует самые сильные навыки, но бой близок к завершению.

**Формула:**

```text
if hp_percent <= final_phase_threshold:
    boss_phase = final
    active_skill_pool = final_phase_skills
```

---

## 20. Посмертная фаза

**Для кого подходит:** нежить, духи, аномалии, демонические существа, мировые боссы, уникальные боссы.

**Описание:** после смерти босс может оставить взрыв, проклятье, волну духов, временную зону или финальное событие.

**Формула:**

```text
if boss_death and post_death_phase_enabled:
    trigger_post_death_event(post_death_event_id)
```

**Ограничение:** посмертная фаза не должна обесценивать победу, но может добавлять риск и атмосферу.

# 8. Благословения

Благословение — положительный эффект, который может временно усиливать моба, игрока, группу, локацию или событие. В контексте мобов благословения можно использовать как редкое усиление противников, награду после победы, эффект священной зоны, результат задания или сезонный бонус.

## 8.1. Общие поля благословения

```yaml
blessing_id: string
blessing_name: string
source_type: zone / quest / achievement / event / item / admin / mob_trait / boss_phase
```

## 51. Благословение силы

**Для чего подходит:** игроки, мобы, рейдовые группы, временные события, священные зоны.

```text
final_max_spirit = base_max_spirit * (1 + spirit_bonus_percent / 100)
```

## 9.2. Где предмет можно использовать

- **В инвентаре** — обычное применение: зелья, книги, свитки, квестовые предметы.
- **В подсумке** — быстрое боевое применение: зелья, бомбы, порошки, метательные предметы.
- **В экипировке** — эффект работает при надевании или даёт активную способность.
- **В особом слоте** — артефакты и редкие реликвии.
- **В локации** — предмет открывает действие только в нужной локации или зоне.
- **В городе** — услуги, торговля, NPC, ратуша, штрафы.
- **Дома** — книги, картины, коллекции, отдых, хранилище.
- **В бою** — расходники и активные предметы с боевыми ограничениями.

# 13. Что НЕ входит в этот файл

В этот файл не входят:

- контр-зелья;
- специальные ослабляющие составы;
- масла против конкретных защит;
- колбы против конкретных атак;
- таблица соответствия «черта → нужное зелье».

Эти элементы должны настраиваться отдельным разделом админ-панели, чтобы их можно было менять без изменения базовых черт мобов.
