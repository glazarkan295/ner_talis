# Рынок в Торговом квартале города Селдар

Итоговая рабочая система для текстовой RPG «Мир Нер-Талис».

## 1. Назначение рынка
Рынок — часть Торговой гильдии в Торговом квартале Селдара. Он используется для безопасной NPC-покупки и NPC-продажи базовых товаров.
- покупка товаров у NPC;
- продажа предметов NPC;
- оборот расходников, простых ингредиентов, инструментов и дешёвых материалов;
- поддержка новичков базовыми товарами;
- стабильная, но не бесконечно выгодная экономическая точка.

## 2. Место в структуре города
Путь игрока: **Селдар → Центральная площадь → Торговый квартал → Торговая гильдия → Рынок**

| Раздел | Роль |
| --- | --- |
| Рынок | NPC-покупка и NPC-продажа обычных городских товаров. |
| Аукцион | Редкие городские лоты, ставки, выставление предметов на срок. |
| Торговый представитель | Контракты, аренда, кредиты и торговые условия. |
| Торговый павильон | Торговля между игроками, основной функционал через сайт. |

## 3. Главное меню рынка
При заходе на рынок игрок видит только 3 кнопки:
- Купить
- Продать
- Торговый павильон

Кнопка **«Торговый павильон»** выводит игрока с рынка в **Торговый павильон**.

## 4. Покупка у NPC
При нажатии кнопки **«Купить»** игроку предоставляется список предметов для покупки. Все цены указаны в **медных монетах за 1 единицу товара**.
### 4.1. Путь покупки
1. Игрок нажимает кнопку «Купить» на рынке.
2. Система показывает список товаров для покупки.
3. Игрок нажимает на нужный товар.
4. Система показывает карточку товара: название, краткое описание, цена покупки за 1 единицу.
5. Под карточкой есть кнопки «Купить» и «Назад».
6. Кнопка «Назад на рынок» возвращает к главному экрану рынка, а из карточки товара кнопка «Назад» возвращает к списку покупки.
7. Кнопка «Купить» спрашивает, сколько единиц товара игрок хочет купить, и просит ввести число сообщением.
8. Игрок вводит число.
9. Система проверяет корректность числа, баланс и место в инвентаре.
10. Если всё корректно, нужная сумма монет списывается с баланса игрока.
11. Купленный предмет добавляется в инвентарь игрока.
12. Если предмет попал в дополнительный слот, это указывается в сообщении покупки.

**Формула списания:** сумма покупки = цена покупки за 1 единицу × количество покупаемых предметов.

**Сообщение успешной покупки:** «Куплено: [название товара] ×[количество]. Потрачено: [сумма монет].»

### 4.2. Проверки при покупке
- введённое значение должно быть целым числом;
- число должно быть больше 0;
- у игрока должно хватать денег;
- товар должен быть в ассортименте;
- должно хватать места в инвентаре или предмет должен корректно перейти в дополнительный слот;
- при попадании в дополнительный слот это должно быть указано в сообщении;
- списание монет и добавление предмета должны выполняться одной операцией.

### 5.1. Примечания к ассортименту
- Стрелы используются только с луками и продаются по 1 штуке.
- Болты используются только с арбалетами. Сейчас цена записана за 1 единицу; позже можно уточнить, поштучная это продажа или пачка.
- Чистая вода используется как единый предмет воды для рынка, еды и алхимии.
- Шахтёрская кирка и топор дровосека являются расходниками-инструментами и имеют по 10 использований.
- Пустые колчаны нужны для хранения боеприпасов: отдельный колчан для стрел лука и отдельный колчан для болтов арбалета.
- Обычные кожаные вещи являются базовой ранней экипировкой и не имеют зачарований.
- Стекляшки являются дешёвой имитацией драгоценных камней и подходят для дешёвой бижутерии, но не должны работать как полноценные камни для сильных зачарований. Цена каждой стекляшки: 400 медных.

## 6. Продажа NPC
Игрок продаёт предметы из своего инвентаря NPC-рынку. **Проданные предметы не появляются в ассортименте NPC.**
1. Игрок нажимает кнопку «Продать» на рынке.
2. Система показывает список всех предметов, находящихся в инвентаре игрока.
3. Игрок нажимает на нужный предмет.
4. Система показывает карточку продажи: описание предмета, количество у игрока, цена продажи за 1 единицу.
5. Под карточкой есть кнопки «Продать» и «Назад».
6. Кнопка «Назад на рынок» возвращает к главному экрану рынка, а из карточки предмета кнопка «Назад» возвращает к списку продажи.
7. Кнопка «Продать» спрашивает, сколько единиц предмета игрок хочет продать, и просит ввести число сообщением.
8. Игрок вводит число.
9. Система проверяет введённое число.
10. Если число корректное, указанное количество предметов удаляется из инвентаря игрока.
11. На баланс игрока начисляется сумма продажи.
12. Проданный предмет не добавляется в ассортимент NPC и не становится доступен другим игрокам через обычный рынок. После успешной продажи игрок возвращается к списку продажи.

**Формула начисления:** сумма продажи = цена продажи за 1 единицу × количество проданных предметов.

**Сообщение успешной продажи:** «Продано: [название предмета] ×[количество]. Получено: [сумма монет].»

### 6.2. Проверки при продаже
- введённое значение должно быть целым числом;
- число должно быть больше 0;
- нельзя продать больше предметов, чем есть в инвентаре;
- предмет должен находиться в инвентаре игрока на момент подтверждения;
- предмет не должен быть экипирован, заблокирован, квестовый или защищённый от продажи;
- цена продажи за 1 единицу должна быть ниже цены покупки у NPC;
- удаление предмета и начисление монет должны выполняться одной операцией;
- проданный предмет не должен записываться в таблицу ассортимента NPC.

## 7. Правило проданных предметов
**Проданные игроком предметы не появляются в ассортименте NPC.**

NPC-рынок выкупает предметы как системный скупщик, но не выставляет их обратно на продажу. Это защищает экономику и не превращает рынок в склад игрока.

## 10. Логические проверки и баланс
- Обычный рынок не должен дублировать портовый рынок.
- Обычный рынок не должен заменять чёрный рынок.
- Обычный рынок не должен заменять аукцион.
- Обычный рынок не должен заменять торговый павильон игроков.
- NPC-продажа не должна быть выгоднее торговли с игроками.
- Цена продажи NPC должна быть ниже цены покупки у NPC.
- Человеческий бонус возврата золота не должен создавать бесконечный заработок на покупке и продаже.
- Высокие качества предметов не должны массово продаваться у обычных NPC.
- Энергия не тратится на обычные городские торговые действия.
- Покупка не должна обходить ограничения инвентаря и дополнительных слотов.
- Продажа важных предметов должна иметь защиту от случайной потери.
- Стекляшки не должны поддерживать полноценные сильные зачарования.
- Обычные кожаные вещи не должны иметь зачарований, потому что зачарование доступно только с эпического качества и выше.
- Колчаны должны быть синхронизированы с правилом боеприпасов: колчан для стрел — для лука, колчан для болтов — для арбалета.

## 11. Что ещё нужно определить позже
- формулу цены продажи NPC для предметов из инвентаря;
- коэффициент выкупа NPC относительно базовой стоимости предмета;
- лимиты покупки и продажи, если они нужны;
- защиту редких, уникальных, квестовых и экипированных предметов от случайной продажи;
- влияние репутации Торговой гильдии на цены;
- влияние долгов на торговлю;
- обновление ассортимента, если рынок будет не полностью фиксированным;
- болты продаются поштучно или пачками;
- нужен ли отдельный быстрый режим покупки/продажи для часто используемых товаров.
