## Точки входа
- `ner_talis_game_project/main.py` — запуск ботов (run_bots: Telegram/VK), восстановление таймеров (recover_runtime_timers), запуск планировщика эффектов (_start_player_effect_scheduler_once).
- `ner_talis_game_project/web_app.py` — FastAPI-приложение сайта (профиль + админка). Защита: TrustedHost, CSP/security-заголовки, rate-limit, защита аплоадов от path-traversal, опц. FORCE_HTTPS/HSTS. **OpenAPI/Swagger (/docs,/redoc,/openapi.json) по умолчанию ВЫКЛЮЧЕНЫ в проде** (ENABLE_API_DOCS=true включает для разработки). Favicon отдаётся (/favicon.svg из dist/public, /favicon.ico → 204) — без 404 в логах. **HTTPS за reverse proxy (ТЗ 23)**: `is_effectively_https()` признаёт запрос защищённым по scheme=https ИЛИ (при TRUST_PROXY_HEADERS и доверенном ближайшем узле) по `X-Forwarded-Proto: https`/`X-Forwarded-Ssl: on`/`Forwarded: proto=https`; `_proxy_ip_trusted()` поддерживает `*`/точные IP/CIDR (та же модель доверия у rate-limit X-Forwarded-For). Uvicorn в timeweb_start.py запускается с `proxy_headers`/`forwarded_allow_ips` из env (UVICORN_PROXY_HEADERS/UVICORN_FORWARDED_ALLOW_IPS). Прямой HTTP без доверенного proxy по-прежнему → 426 «HTTPS required». Переменные — в README_TIMEWEB.md и .env.timeweb.postgresql.example.
- `ner_talis_game_project/site_api.py` — API профиля игрока (frontend_profile, инвентарь/навыки/характеристики/предметы/edit-field). Профиль V2 отдаёт status/warnings, только личные места рейтингов, сервисы, гильдию, открытые репутации, участие/очки/личное место эвентов и опубликованную раскладку. React-профиль применяет вкладки/блоки/тему и показывает репутацию/эвенты без формул и скрытых значений.
- `ner_talis_game_project/admin_panel_api.py` — API админ-панели V1 (роуты /api/admin/...).

## Важные инварианты (легко сломать)
- Боты ПОСЛЕ действия пересохраняют ИСХОДНЫЙ объект игрока (handlers/city.py:~113, vk). Поэтому claim-перезагрузки (события/рыбалка) надо синхронизировать В исходный объект (`player.clear(); player.update(claimed)`), иначе изменения затрутся.
- Тесты запускать из корня репозитория (иначе ложные падения проверок иконок `web/public/...`).
- Деньги хранятся в меди (64-бит) — награды монетами ограничены по медному эквиваленту (admin_panel_service.MAX_REWARD_MONEY_COPPER).
